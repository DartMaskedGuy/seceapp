Secretary App


The secretary app is a great assistance for the recording, editing and saving activities taken place in a meeting.

Features:

  1. Converts spoken words into text
  2. Auto Save Feature
  3. A copy button that copies the text
  4. A save button that saves the converted speech to text in a memory
  5. A start button to start the recording
  6. A pause button to pause the recording
  7. A stop button to end recording
  8. An Email forwarder that sends recorded text to an email
